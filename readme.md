# bredbandskollen-cli

Measure your internet speed from the command line using Bredbandskollen.

![](assets/screenshot.png)

## Install

```bash
$ npm install -g bredbandskollen-cli
```

## CLI

```
Usage
	  $ bredbandskollen

	Options
		--json  JSON output

	Examples
	  $ bredbandskollen --json
```
